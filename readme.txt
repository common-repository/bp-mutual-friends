== Description ==

List users' mutual friends in BuddyPress easily. One click install and setup.


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'BP Mutual Friends'
3. Activate BP Mutual Friends from your Plugins page. 

= From WordPress.org =

1. Download BP Mutual Friends.
2. Upload the 'bp-mutual-friends' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate BP Mutual Friends from your Plugins page.
